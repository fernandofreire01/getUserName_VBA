## Get User Name - VBA

Script retorna o nome do usuário atual do SO (sistema operacional). 
Comandos foram testados na Macro do Excel.


### Código:
```vbscript
Sub test_user()

MsgBox "Current user is " & Application.UserName

End Sub
```
